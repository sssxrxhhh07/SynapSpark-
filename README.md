# 📈 Mathematical Models Used

## 1. Time Series Forecasting

Predict future electricity demand.

Methods:

* ARIMA
* SARIMA
* Seasonal Analysis

Formula:

y(t)=Trend+Seasonality+Residual

---

## 2. Risk Prediction Score

Risk Index:

```text
Risk =
0.4 × Load%
+
0.3 × Temperature
+
0.2 × Outages
+
0.1 × Overloads
```

Classifications:

* Safe
* Warning
* Danger

---

## 3. Load Redistribution Optimizer

Objective:

```text
Minimize

Σ(Load - Capacity)²
```

Result:

* Balanced grid
* Reduced overloads
* Lower outage probability

---

## 4. Transformer Failure Prediction

Uses:

* Weibull Distribution
* Survival Analysis

Formula:

```text
R(t)=e^(-(t/η)^β)
```

Predicts:

* Remaining Useful Life
* Failure Probability

---

## 5. Graph Theory Blackout Simulator

Algorithms:

* Max Flow
* Min Cut
* Dijkstra

Used for:

* Cascade Failure Analysis
* Power Rerouting
* Recovery Planning

---

## 6. FFT Grid DNA Analyzer

Uses:

Fast Fourier Transform

Converts:

```text
Load Curve
     ↓
Frequency Signature
```

Applications:

* Pattern Recognition
* Electricity Theft Detection
* Anomaly Detection

---

## 7. Carbon Optimization

Linear Programming

Objective:

```text
Minimize Carbon Emissions
```

Constraints:

```text
Supply ≥ Demand
```

Output:

Optimal Energy Mix
